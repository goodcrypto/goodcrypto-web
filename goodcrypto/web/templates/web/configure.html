Configure Browsers to Use GoodCrypto

For each user:

  1. Configure their browser to use the GoodCrypto web proxy. If you have any internal web servers, then you probably want to exclude those servers from using the proxy.

  2. Add the GoodCrypto web certificate to your organization's certificate authority white list, or import the web certificate for each browser.

Each person that uses the GoodCrypto Web proxy reduces tracking of their online activity and the possibility of their computer being cracked.
