Firefox

  1. Select Preferences from the Edit menu.
    If you're running Firefox on Mac, then select Preferences... from the Firefox menu.
  2. Click the Advanced icon at the top of the dialog box.
  3. Click the Encryption tab, and then the Authorities tab.
  4. Click the View certificates button.
  5. Click the Import button.
  6. Locate the file that you saved in Step 1 of these instructions and hightlight the filename and Open it.
  7. If Firefox pops up the warning, "This certificate is already installed as a certificate authority", then click here
  8. Another dialog will pop up that asks, "Do you want to trust "CA Cert Signing Authority" for the following purposes?".
  9. Add a check mark in the Trust this CA to identify web sites box and click the OK button.
  10. Close all the windows and dialog boxes until you have returned to your main browser window.

If Firefox pops up the warning, "This certificate is already installed as a certificate authority", then


  1. Click the OK button
  2. Highlight the ca.mitm.com item in the list
  3. Click the Delete or Distrust button and confirm when prompted
  4. Resume the procedure by restarting with Click the Import button step.
